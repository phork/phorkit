<PageTitle src="compositions/Pagination" title="Pagination" />

The pagination is a wrapper for the [ButtonGroup](../components/ButtonGroup). It can use either `ButtonGroup` styles or custom styles on top of an unstyled `ButtonGroup`.

## Pagination

### Pagination as buttons

<Playground>
  <ThemeWrapper>
    <StateWrapper initialState={8}>
      {({ state: currentPage, setState: setCurrentPage }) => (
        <Pagination
          withIcons
          withPageLinks
          withPreviousAndNext
          color="primary"
          onChangePage={(event, pageTo) => setCurrentPage(pageTo)}
          page={currentPage}
          pageLinks={6}
          pageSize={10}
          totalItems={300}
        />
      )}
    </StateWrapper>
  </ThemeWrapper>
</Playground>

### Pagination as links

_Note that the links will cause the page to reload so the state will appear unchanged._

<Playground>
  <ThemeWrapper>
    <StateWrapper initialState={8}>
      {({ state: currentPage }) => (
        <Pagination
          withIcons
          withPageLinks
          withPreviousAndNext
          color="primary"
          getHref={page => `#page${page}`}
          page={currentPage}
          pageLinks={6}
          pageSize={10}
          totalItems={300}
        />
      )}
    </StateWrapper>
  </ThemeWrapper>
</Playground>
